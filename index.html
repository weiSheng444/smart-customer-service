常见问题

一、订单与支付类

Q1: 下单后如何修改收货地址/商品规格？

+

✅ 解决方案：

  • 若订单状态为「待付款」：直接取消订单重新下单
  • 若已付款但未发货：立即联系在线客服（提供订单号）
  • 已发货订单：联系物流公司尝试拦截（需承担可能产生的运费）

Q2: 支付失败怎么办？

+

✅ 解决方案：

  • 检查网络连接是否正常
  • 确认银行卡余额充足
  • 确认支付限额是否超出
  • 尝试更换支付方式
  • 如仍无法解决，请联系银行客服

Q3: 如何申请发票？

+

✅ 操作步骤：

  1. 登录账户进入"我的订单"
  2. 找到需要开票的订单
  3. 点击"申请发票"按钮
  4. 填写发票信息（抬头、税号等）
  5. 提交申请，1-3个工作日内开具

注：电子发票将发送至注册邮箱

Q4: 优惠券如何使用？

+

✅ 使用说明：

  • 结算页面选择可用优惠券
  • 注意使用条件和有效期
  • 每笔订单限用一张优惠券
  • 部分商品可能不参与优惠
  • 优惠券不可叠加使用

账户问题

账户注册与登录

如何注册新账户？

1. 使用手机号注册

  • 点击右上角"注册"按钮
  • 输入手机号并获取验证码
  • 设置登录密码（8-20位字符）
  • 完善基本信息即可

2. 第三方账号快捷注册

  • 支持微信一键登录
  • 支持支付宝账号授权

账户安全设置

建议开启以下安全设置：

  • 登录密码：定期更换
  • 支付密码：设置独立密码
  • 手机验证：绑定常用手机
  • 双重认证：增加登录安全性

忘记密码怎么办？

✅ 重置密码步骤：

  1. 点击登录页面的"忘记密码"
  2. 输入注册手机号获取验证码
  3. 验证身份后设置新密码
  4. 建议使用8-20位包含字母和数字的组合

注：如无法接收验证码，请联系客服

如何修改绑定手机号？

✅ 修改流程：

  • 登录账户进入安全中心
  • 选择"修改绑定手机"
  • 验证原手机号
  • 输入新手机号并验证
  • 修改成功后原手机号将解绑

注意：修改后需重新绑定支付方式

账户被锁定怎么办？

✅ 解锁方法：

  • 连续输错密码5次将触发锁定
  • 24小时后自动解锁
  • 或通过手机验证立即解锁
  • 建议开启双重认证提高安全性

退款政策

退款规则

🕒 退款时效

  • 未发货订单：1-3个工作日
  • 已发货订单：7-15个工作日
  • 仅退款：3个工作日内

📦 退货要求

  • 商品及包装保持完好
  • 不影响二次销售
  • 附带原始购物凭证

❌ 不支持退款情况

  • 人为损坏商品
  • 已使用或破损
  • 超过7天退换期
  • 定制类商品
